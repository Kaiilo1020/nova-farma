=====================================================
CARPETA DE LIBRERÍAS - Nova Farma
=====================================================

Esta carpeta debe contener el driver JDBC de PostgreSQL.

ARCHIVO REQUERIDO:
------------------
postgresql-42.7.8.jar (o versión superior)

DESCARGA:
---------
https://jdbc.postgresql.org/download/

INSTRUCCIONES:
--------------
1. Visita el enlace de descarga
2. Descarga la última versión estable del driver JDBC
3. Guarda el archivo .jar en esta carpeta (lib/)
4. Verifica que el nombre coincida con el usado en compile.bat/compile.sh

VERSIONES COMPATIBLES:
----------------------
- postgresql-42.7.8.jar (recomendada - versión actual del proyecto)
- postgresql-42.7.1.jar
- postgresql-42.6.0.jar
- Cualquier versión 42.x.x

NOTA:
-----
Si usas una versión diferente, actualiza los scripts de compilación:
- compile.bat (Windows)
- compile.sh (Linux/Mac)

Cambia "postgresql-42.7.8.jar" por el nombre de tu archivo.
